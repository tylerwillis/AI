# Experiments

Small, self-contained projects exploring what's possible with AI tools. Each experiment is a snapshot of something that worked (or didn't) -- not polished production code, but useful as reference and inspiration.

## [Blog Image Generator](blog-image-gen/)

Local-first tool for generating blog header images using Gemini's Imagen 3. Style presets, aspect ratios, reference images, iterative refinement, and a gallery with SQLite history. Built with Bun in a single Claude Code session.

## [SimConv](simconv/)

AI-powered "pre-flight simulator" for high-stakes conversations. Create personas of people in your network, then practice pitches, test email drafts, and rehearse difficult conversations with AI digital twins.
