# AI

A collection of prompts, guides, and tools for working with AI by [Tyler Willis](https://x.com/tylerwillis).

## [Prompts](prompts/)

Standalone prompts for LLM-assisted tasks -- deck reviews, company deep-research, and more.

## [Guides](guides/)

Step-by-step workflows for AI-assisted content creation -- turning video recordings into presentation decks, generating images with iterative refinement, and more.

## [Skills](skills/)

Reusable Claude Code skills -- specialized instructions that teach an AI agent how to use a specific API, follow a workflow, or apply conventions.

## [DeepWork Jobs](deepwork-jobs/)

Portable multi-step workflow definitions with quality gates. Drop into any project and run with DeepWork.

## [Experiments](experiments/)

Small, self-contained projects exploring what's possible with AI tools. Reference and inspiration, not production code.
