# Skills

Reusable [Claude Code skills](https://docs.anthropic.com/en/docs/claude-code/skills) -- specialized instructions that extend what an AI coding agent can do. Each skill teaches the agent a specific capability: how to use an API, follow a workflow, or apply a set of conventions.

Skills are markdown files that get loaded into the agent's context when triggered. They're like giving the agent a reference card for a specific task.

## [QA](qa.md)

Adversarial website quality assurance. Systematically validates content through multiple passes: site completeness, placeholder detection, source verification, and credibility review. Built to catch the specific failure modes of LLM-generated sites.

## [Frontend Design](frontend-design.md)

Official Anthropic plugin that generates distinctive, production-grade frontend interfaces. Overrides the tendency toward generic "AI-generated" aesthetics with opinionated design direction.
